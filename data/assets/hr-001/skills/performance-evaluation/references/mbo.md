# 目标管理法（MBO）考核规则参考

## 概述

目标管理法（Management by Objectives，MBO）由管理学大师彼得·德鲁克于1954年在《管理的实践》中首次提出。MBO的核心思想是：组织的高层管理者与下属共同制定目标，通过目标将组织的整体目标层层分解为各部门和个人的具体目标，形成目标体系，并以目标完成情况作为考核和评价的依据。MBO强调目标的自上而下分解与自下而上承诺的统一，注重目标达成过程中的参与感和责任感。

## MBO与OKR的区别

MBO和OKR都强调目标驱动，但存在关键差异，准确区分有助于正确选择考核方式：

| 维度 | MBO | OKR |
|------|-----|-----|
| 目标来源 | 上下级共同协商确定 | 员工自主提出为主，上级对齐 |
| 目标性质 | 务实可达成的目标 | 鼓励挑战性甚至不可完全达成的目标 |
| 考核结果应用 | 直接关联薪酬、晋升 | 不直接挂钩薪酬，侧重发展反馈 |
| 时间周期 | 通常年度或半年度 | 通常季度，节奏更快 |
| 公开程度 | 通常仅上下级之间可见 | 全员公开透明 |
| 评分逻辑 | 目标达成率越高越好 | 0.6-0.7为理想，1.0可能目标过低 |

## MBO目标设定规范

### 目标设定原则

1. **上下协商：** 目标由上级提出方向、下级提出具体方案，双方协商确定，确保目标的合理性和承诺度
2. **SMART原则：** 目标必须具体（Specific）、可衡量（Measurable）、可达成（Attainable）、相关性（Relevant）、有时限（Time-bound）
3. **目标关联：** 个人目标必须与部门/组织目标形成明确的支撑关系
4. **数量适中：** 每个岗位设定3-6个核心目标，避免过多导致精力分散
5. **优先级排序：** 目标需标注优先级（P1/P2/P3），确保关键目标得到重点保障

### 目标分解方法

```
组织战略目标
    └── 部门目标（承接组织战略）
        └── 团队目标（承接部门目标）
            └── 个人目标（承接团队目标）
```

每个层级目标必须满足：
- 与上层目标有明确的支撑关系
- 有可量化的衡量标准
- 有明确的完成时限
- 有可识别的关键里程碑

## 评估维度

### 1. 目标达成度（权重：50%）

评估各项目标的实际完成情况，是MBO考核的核心维度。

**评分标准：**

| 目标达成率 | 评分 | 描述 |
|-----------|------|------|
| ≥110% | 5分 | 超额完成目标，表现卓越 |
| 100%-109% | 4分 | 完全达成目标，表现优秀 |
| 85%-99% | 3分 | 基本达成目标，表现合格 |
| 70%-84% | 2分 | 部分达成目标，需改进 |
| <70% | 1分 | 严重未达成目标，需全面整改 |

目标达成率计算：
```
目标达成率 = 实际完成值 ÷ 目标值 × 100%
```

**优先级加权：** 不同优先级的目标在综合达成度中的权重不同：

| 优先级 | 权重系数 | 说明 |
|--------|---------|------|
| P1（核心目标） | 1.5 | 必须完成的关键目标 |
| P2（重要目标） | 1.0 | 重要但非最关键的目标 |
| P3（一般目标） | 0.5 | 期望完成但非刚性要求 |

```
加权目标达成度 = Σ（目标i达成率 × 目标i优先级权重系数）÷ Σ（目标i优先级权重系数）
```

### 2. 目标质量（权重：20%）

评估目标设定的合理性和挑战性。

| 评估项 | 评分标准 |
|--------|---------|
| 目标清晰度 | 目标描述明确、无歧义，衡量标准清晰（5-1分） |
| 目标挑战性 | 目标需要付出显著努力才能达成（5-1分） |
| 目标对齐度 | 个人目标与组织/部门目标紧密关联（5-1分） |
| 目标可衡量性 | 所有目标都有明确的量化衡量标准（5-1分） |

### 3. 过程管理（权重：15%）

评估目标执行过程中的管理表现。

| 评估项 | 评分标准 |
|--------|---------|
| 进度跟踪 | 定期（至少月度）检查目标进度，及时发现问题（5-1分） |
| 阶段汇报 | 按约定周期向上级汇报目标进展，信息透明（5-1分） |
| 偏差调整 | 目标偏离时能及时分析原因并采取纠正措施（5-1分） |
| 资源协调 | 主动协调所需资源，保障目标推进（5-1分） |

### 4. 能力与态度（权重：15%）

评估目标执行过程中展现的能力素质和工作态度。

| 评估项 | 评分标准 |
|--------|---------|
| 主动性 | 不等不靠，主动推进工作（5-1分） |
| 协作精神 | 积极配合他人，推动团队目标达成（5-1分） |
| 问题解决 | 遇到困难能独立分析并有效解决（5-1分） |
| 学习成长 | 在目标执行中不断学习和改进（5-1分） |

## 综合得分计算方法

### 第一步：计算各维度得分

```
目标达成度得分 = 加权目标达成率对应评分
目标质量得分 = Σ（目标质量各评估项得分）÷ 评估项数量
过程管理得分 = Σ（过程管理各评估项得分）÷ 评估项数量
能力与态度得分 = Σ（能力与态度各评估项得分）÷ 评估项数量
```

### 第二步：计算综合得分

```
综合得分 = 目标达成度得分 × 50% + 目标质量得分 × 20% 
         + 过程管理得分 × 15% + 能力与态度得分 × 15%
```

## 绩效等级划分

| 等级 | 得分范围 | 描述 |
|------|---------|------|
| S（卓越） | 4.5 - 5.0 | 所有核心目标超额完成，过程管理优秀，能力突出 |
| A（优秀） | 3.8 - 4.4 | 大部分目标达成或超额完成，过程管理良好 |
| B（良好） | 3.0 - 3.7 | 基本达成各项目标，过程管理正常 |
| C（待改进） | 2.0 - 2.9 | 部分目标未达成，过程管理存在不足 |
| D（不合格） | 1.0 - 1.9 | 多项目标未达成，需全面整改 |

## MBO特有的报告要求

1. **目标清单与达成分析：** 逐项列出每项目标的描述、优先级、目标值、实际值、达成率和得分，以表格形式清晰呈现
2. **目标分解链路图：** 展示个人目标→团队目标→部门目标→组织目标的分解链路，体现目标对齐关系
3. **里程碑回顾：** 对每项目标的关键里程碑完成情况进行回顾，标注提前/按时/延迟状态
4. **目标协商记录：** 记录目标设定时的协商过程和最终承诺，确保考核的公正性
5. **下期目标建议：** 基于本期目标执行情况，对下期目标的设定方向、挑战性和数量提出建议
6. **过程管理评价：** 单独设立"过程管理评价"小节，评估目标执行过程中的跟踪、汇报和调整情况
