# 关键绩效指标（KPI）考核规则参考

## 概述

关键绩效指标（Key Performance Indicator，KPI）是一种通过对组织内部流程的输入端、输出端的关键参数进行设置、取样、计算、分析，衡量流程绩效的一种目标式量化管理指标。KPI考核的核心理念是：把企业的战略目标分解为可操作的工作目标，用可量化的指标来衡量员工的工作表现。

## KPI设计原则（SMART原则）

设计KPI指标时必须遵循SMART原则：

| 原则 | 含义 | 示例 |
|------|------|------|
| S - Specific（具体的） | 指标必须明确具体 | "月销售额达到100万元"而非"提高销售" |
| M - Measurable（可衡量的） | 指标必须可量化 | "客户满意度≥90分"而非"让客户满意" |
| A - Attainable（可达成的） | 指标经过努力可实现 | 基于历史数据和合理增长设定 |
| R - Relevant（相关的） | 指标与岗位职责强相关 | 销售岗位考核销售业绩，非考勤率 |
| T - Time-bound（有时限的） | 指标有明确的时间范围 | "本季度内完成"而非"尽快完成" |

## KPI指标分类

### 1. 定量指标

可直接用数字衡量的指标，是KPI考核的核心。

**评分标准：**

| 完成率 | 评分 |
|--------|------|
| ≥120% | 5分（卓越） |
| 100%-119% | 4分（优秀） |
| 80%-99% | 3分（合格） |
| 60%-79% | 2分（待改进） |
| <60% | 1分（不合格） |

完成率计算公式：
```
完成率 = 实际完成值 ÷ 目标值 × 100%
```

### 2. 定性指标

无法直接量化但重要的指标，需通过行为描述或等级评定来评估。

**评分标准：**

| 等级 | 评分 | 行为描述 |
|------|------|---------|
| 卓越 | 5分 | 持续超出期望，可作为标杆 |
| 优秀 | 4分 | 经常超出期望，表现突出 |
| 合格 | 3分 | 达到岗位要求，表现稳定 |
| 待改进 | 2分 | 部分未达要求，需加强 |
| 不合格 | 1分 | 明显未达要求，需立即改善 |

## 权重分配规则

1. 单个KPI指标权重一般不低于5%，不高于40%
2. 核心业务指标权重应高于支撑性指标
3. KPI指标总数建议控制在5-8个
4. 所有指标权重之和必须等于100%
5. 如有否决性指标（如安全事故），不设权重，触发时绩效等级直接降级

## 综合得分计算方法

```
综合得分 = Σ（KPI指标i得分 × KPI指标i权重）
```

对于定量指标：
```
指标得分 = 完成率对应评分等级的分数
```

对于定性指标：
```
指标得分 = 行为等级对应评分的分数
```

## 绩效等级划分

| 等级 | 得分范围 | 描述 | 绩效系数 |
|------|---------|------|---------|
| S（卓越） | 4.5 - 5.0 | 全面超越目标 | 1.3 |
| A（优秀） | 3.8 - 4.4 | 大部分指标超额完成 | 1.1 |
| B（良好） | 3.0 - 3.7 | 基本达成目标 | 1.0 |
| C（待改进） | 2.0 - 2.9 | 多项指标未达标 | 0.8 |
| D（不合格） | 1.0 - 1.9 | 严重未达标 | 0.5 |

## KPI特有的报告要求

1. **指标达成分析：** 逐项列出每个KPI的目标值、实际值、完成率和得分，用表格清晰呈现
2. **趋势对比：** 如有历史数据，对比上期得分变化趋势，用↑↓→标注
3. **归因分析：** 对未达标指标进行原因分析，区分客观因素和主观因素
4. **目标调整建议：** 基于本期实际表现，对下期目标值的合理性给出建议
5. **数据来源标注：** 每项指标必须标注数据来源，确保评估的可追溯性和客观性
