# 薪酬全绩效（KSF）考核规则参考

## 概述

薪酬全绩效（Key Success Factors，KSF）是一种将薪酬与绩效全面融合的考核分配模式，由李太林在传统绩效管理基础上创新提出。KSF的核心思想是：将员工的薪酬结构重新设计，把固定薪酬的一部分转化为与绩效挂钩的浮动薪酬，通过识别影响岗位成功的关键成功因子，为每个因子设定目标值和平衡点，实现"员工加薪、企业增效"的双赢局面。KSF强调利益共同体，让员工从"为老板打工"转变为"为自己创造价值"。

## KSF与KPI的区别

KSF和KPI都关注量化指标，但理念和应用方式有本质不同：

| 维度 | KSF | KPI |
|------|-----|-----|
| 核心目的 | 员工价值创造与收入增长的统一 | 目标达成与绩效评价 |
| 结果应用 | 直接决定薪酬增减 | 评价等级，间接影响薪酬 |
| 指标性质 | 关注正向激励和增值分配 | 关注目标完成度和差距分析 |
| 评分逻辑 | 以平衡点为基准，超出有奖、低于有扣 | 以目标值为基准，计算完成率 |
| 员工感受 | 多劳多得，主动追求更高绩效 | 被考核被约束，被动应对 |
| 薪酬结构 | 固定+浮动，浮动比例较大 | 通常固定为主，绩效为辅 |

## KSF核心概念

### 1. 关键成功因子（KSF指标）

KSF指标是影响岗位价值产出的最核心因素，每个岗位识别6-8个KSF指标。

**KSF指标选取原则：**
- **价值导向：** 选取对岗位产出价值影响最大的因素
- **可量化：** 指标必须有明确的量化衡量方式
- **可驱动：** 员工通过自身努力可以直接影响该指标的结果
- **平衡性：** 指标之间应涵盖数量、质量、效率、成本等不同维度

### 2. 平衡点

平衡点是KSF的关键创新，指该指标在过去一段时期（通常6-12个月）的平均水平或合理均值。平衡点是薪酬不增不减的临界值。

**平衡点设定方法：**

| 方法 | 适用场景 | 计算方式 |
|------|---------|---------|
| 历史均值法 | 数据稳定的指标 | 过去6-12个月实际值的平均值 |
| 行业标杆法 | 缺乏内部历史数据 | 同行业同岗位的合理水平 |
| 协商确定法 | 新设岗位或新指标 | 上下级协商确定一个合理基准值 |

### 3. 分配规则

KSF的核心机制是围绕平衡点的增值分配：

```
当指标实际值 > 平衡点时 → 正向激励：超出部分按约定比例增加薪酬
当指标实际值 = 平衡点时 → 基准状态：该指标薪酬不变
当指标实际值 < 平衡点时 → 负向调节：低于部分按约定比例扣减薪酬
```

## 薪酬计算方法

### 第一步：确定薪酬结构

```
月度薪酬 = 基本工资 + Σ（各KSF指标浮动薪酬）
```

其中：
- 基本工资：固定部分，保障基本生活，通常占原总薪酬的50%-70%
- KSF浮动薪酬：各指标根据平衡点和分配规则计算出的增减金额

### 第二步：计算各KSF指标的浮动薪酬

```
若实际值 > 平衡点：
  该指标浮动薪酬 = （实际值 - 平衡点）× 单位激励

若实际值 = 平衡点：
  该指标浮动薪酬 = 0

若实际值 < 平衡点：
  该指标浮动薪酬 = -（平衡点 - 实际值）× 单位扣减
```

### 第三步：汇总计算

```
最终月度薪酬 = 基本工资 + Σ（各KSF指标浮动薪酬i）
```

> **保底规则：** 最终月度薪酬不得低于当地最低工资标准。如计算结果低于最低标准，按最低标准发放。

> **封顶规则：** 可根据企业情况设定浮动薪酬上限，通常不超过基本工资的100%。

## 综合绩效评分

虽然KSF的核心是薪酬计算，但为生成绩效评估报告，需要将KSF结果转化为绩效评分。

### 评分转化方法

```
KSF指标得分 = （实际值 - 平衡点）÷ 平衡点 × 100%
```

转化为5分制：

| KSF指标得分范围 | 5分制评分 | 描述 |
|---------------|---------|------|
| > 20% | 5分 | 显著超越平衡点 |
| 10% - 20% | 4分 | 明显超越平衡点 |
| 0% - 9% | 3分 | 达到或略超平衡点 |
| -10% - -1% | 2分 | 略低于平衡点 |
| < -10% | 1分 | 明显低于平衡点 |

### 综合得分计算

```
综合得分 = Σ（KSF指标i评分 × KSF指标i权重）
```

## 绩效等级划分

| 等级 | 得分范围 | 描述 | 薪酬影响 |
|------|---------|------|---------|
| S（卓越） | 4.5 - 5.0 | 多数指标显著超越平衡点 | 薪酬大幅增长（+30%以上浮动） |
| A（优秀） | 3.8 - 4.4 | 大部分指标超越平衡点 | 薪酬明显增长（+15%-30%浮动） |
| B（良好） | 3.0 - 3.7 | 指标基本达到平衡点 | 薪酬小幅增长（+5%-15%浮动） |
| C（待改进） | 2.0 - 2.9 | 多项指标低于平衡点 | 薪酬基本持平或微降 |
| D（不合格） | 1.0 - 1.9 | 大部分指标显著低于平衡点 | 薪酬下降，需启动改善计划 |

## KSF特有的报告要求

1. **KSF指标与平衡点一览表：** 列出所有KSF指标、平衡点、单位激励/扣减标准、权重，以表格形式清晰呈现
2. **薪酬测算明细：** 逐项计算各KSF指标的浮动薪酬增减金额，最终汇总得出实际薪酬变动
3. **平衡点达成分析：** 对比各指标实际值与平衡点的差异，分析超越或低于平衡点的原因
4. **增值贡献分析：** 计算员工为组织创造的增量价值，体现"员工加薪、企业增效"的双赢逻辑
5. **激励效果评估：** 评估KSF方案对员工积极性的激发效果，分析是否存在激励不足或过度激励的情况
6. **平衡点调整建议：** 基于本期数据，建议下期各指标的平衡点是否需要调整，以及调整方向和幅度
7. **薪酬安全线说明：** 在报告末尾注明保底和封顶规则，确保薪酬计算的合规性
