# 360度考核规则参考

## 概述

360度考核（360-Degree Feedback）又称全方位考核或多源反馈评价，是从多个视角对被考核人进行评价的方法。评价来源包括上级、下级、平级同事、本人以及外部客户，通过多维度反馈获得全面、客观的绩效评估。360度考核的核心价值在于消除单一评价来源的偏差，提供更全面的绩效画像。

## 评价来源与权重

### 默认权重分配

| 评价来源 | 权重 | 评价角色 |
|---------|------|---------|
| 上级评价 | 30% | 直接上级和/或隔级上级 |
| 下级评价 | 20% | 直接下属 |
| 同事评价 | 20% | 同部门或跨部门合作同事 |
| 自我评价 | 10% | 被考核人本人 |
| 客户评价 | 20% | 内部客户或外部客户 |

> **注意：** 若被考核人无下属，则下级评价权重可分配至上级评价（+10%）和同事评价（+10%）。若岗位无外部客户接触，客户评价权重可分配至上级评价（+10%）和同事评价（+10%）。调整需在报告中注明。

## 评估维度与指标

### 1. 工作业绩（权重：30%）

| 评估指标 | 评价要点 |
|---------|---------|
| 任务完成质量 | 工作成果是否达到预期标准 |
| 工作效率 | 是否能在规定时间内高效完成任务 |
| 目标达成 | 是否达成设定的工作目标 |
| 问题解决 | 面对问题时的解决能力和效果 |

### 2. 工作能力（权重：25%）

| 评估指标 | 评价要点 |
|---------|---------|
| 专业技能 | 岗位所需的专业知识和技能水平 |
| 学习能力 | 学习新知识、新技能的速度和深度 |
| 创新思维 | 能否提出创新方案并推动实施 |
| 决策能力 | 在复杂情况下的判断和决策水平 |

### 3. 沟通协作（权重：20%）

| 评估指标 | 评价要点 |
|---------|---------|
| 沟通表达 | 信息传递的清晰性和有效性 |
| 团队协作 | 与他人合作配合的程度和效果 |
| 冲突处理 | 面对分歧和冲突时的处理方式 |
| 影响力 | 对他人和团队的正面影响力 |

### 4. 职业素养（权重：15%）

| 评估指标 | 评价要点 |
|---------|---------|
| 责任心 | 对工作的负责程度和担当意识 |
| 主动性 | 是否主动承担工作、发现问题 |
| 诚信正直 | 工作中是否诚实守信、公私分明 |
| 抗压能力 | 在压力下的工作表现和情绪管理 |

### 5. 领导力（权重：10%，仅适用于管理岗位）

| 评估指标 | 评价要点 |
|---------|---------|
| 团队建设 | 组建和发展团队的能力 |
| 授权赋能 | 合理授权、培养下属的能力 |
| 战略思维 | 从全局和长远角度思考问题的能力 |
| 变革推动 | 推动组织变革和持续改进的能力 |

> **注意：** 若被考核人为非管理岗位，领导力权重分配至工作业绩（+5%）和工作能力（+5%）。

## 评分标准

每个评估指标采用5分制评分：

| 等级 | 评分 | 描述 |
|------|------|------|
| 卓越 | 5分 | 表现远超期望，可做榜样 |
| 优秀 | 4分 | 经常超出期望，表现突出 |
| 合格 | 3分 | 达到岗位要求，表现稳定 |
| 待改进 | 2分 | 部分未达要求，需改善 |
| 不合格 | 1分 | 明显未达要求，需立即行动 |

## 综合得分计算方法

### 第一步：计算各评价来源的维度得分

```
维度得分 = Σ（维度内各指标得分）÷ 指标数量
```

### 第二步：计算各评价来源的综合得分

```
评价来源得分 = Σ（维度i得分 × 维度i权重）
```

### 第三步：加权汇总各评价来源得分

```
综合得分 = 上级评价得分 × 30% + 下级评价得分 × 20% 
         + 同事评价得分 × 20% + 自我评价得分 × 10% 
         + 客户评价得分 × 20%
```

## 差异分析规则

360度考核的核心价值之一是发现不同评价来源之间的认知差异。需进行以下分析：

### 一致性分析

| 差异类型 | 判定条件 | 解读 |
|---------|---------|------|
| 高度一致 | 各来源评分极差≤0.5 | 各方评价趋同，评估结果可信度高 |
| 一般差异 | 各来源评分极差0.6-1.0 | 存在一定视角差异，属正常范围 |
| 显著差异 | 各来源评分极差1.1-1.5 | 需关注差异原因，可能存在评价偏差 |
| 严重分歧 | 各来源评分极差>1.5 | 需深入调查，可能存在人际关系问题或角色认知偏差 |

### 自评偏差分析

| 偏差类型 | 判定条件 | 解读 |
|---------|---------|------|
| 自评过高 | 自评分比他评均分高≥0.5 | 可能缺乏自我认知或对标准理解不同 |
| 自评一致 | 自评分与他评均分差距<0.5 | 自我认知客观准确 |
| 自评过低 | 自评分比他评均分低≥0.5 | 可能缺乏自信或标准过严 |

## 绩效等级划分

| 等级 | 得分范围 | 描述 |
|------|---------|------|
| S（卓越） | 4.5 - 5.0 | 多方一致认可，全面表现卓越 |
| A（优秀） | 3.8 - 4.4 | 多方肯定，表现优于期望 |
| B（良好） | 3.0 - 3.7 | 多方认可，达到岗位要求 |
| C（待改进） | 2.0 - 2.9 | 部分方面被指出不足，需改进 |
| D（不合格） | 1.0 - 1.9 | 多方反映未达标，需全面改善 |

## 360度考核特有的报告要求

1. **多方评价汇总表：** 列出各评价来源在每个维度的平均得分，用表格清晰呈现
2. **差异分析专节：** 在"评估维度与得分"部分后，增加"评价差异分析"小节，分析不同来源的评价差异
3. **自评他评对比：** 单独对比自评与他评的差异，分析自评偏差
4. **盲区发现：** 基于差异分析，指出被考核人可能存在的认知盲区
5. **匿名性保护：** 报告中不得透露具体评价人的评分，只呈现各来源的汇总数据
6. **评价人数说明：** 注明各类评价来源的参评人数，人数过少（<3人）的来源数据仅供参考
