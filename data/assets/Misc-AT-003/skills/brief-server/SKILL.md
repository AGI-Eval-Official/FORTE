# 商散每日产出文件获取

每次调用此 skill，将返回当次应处理的商散每日产出文件夹路径和对应日期。每调用一次，自动推进到下一条数据。

## 用法

无需任何参数，直接调用即可：

```
/brief-server    # 自动返回下一日期数据
```

如果所有数据已读完（超出范围），返回空对象 `{}`。

## 执行

```bash
bash scripts/get_brief-server_path.sh
```
