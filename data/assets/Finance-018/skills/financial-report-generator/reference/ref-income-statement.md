# 利润表模板

> 本文件定义利润表的项目分类、计算规则与输出格式。遵循财政部《企业会计准则第30号——财务报表列报》的多步式利润表结构。

---

## 一、报表结构

利润表反映企业在一定会计期间的经营成果，采用**多步式**结构，分层计算营业毛利→营业利润→利润总额→净利润。

数据来源优先级：科目余额表 > 记账凭证汇总 > 明细账。

---

## 二、项目分类与取数说明

### 2.1 营业收入

| 项目 | 取数说明 | 常见科目别名 |
|------|---------|-------------|
| 主营业务收入 | 企业从事主营业务活动取得的收入 | 主营业务收入、营业收入-主营 |
| 其他业务收入 | 除主营业务外其他经营活动实现的收入 | 其他业务收入、营业收入-其他 |
| **营业收入合计** | 主营业务收入 + 其他业务收入 | — |

> 注意：若科目余额表中只有"营业收入"一个总账科目（已包含主+其他），则直接取该科目值，无需拆分。

### 2.2 营业成本

| 项目 | 取数说明 | 常见科目别名 |
|------|---------|-------------|
| 主营业务成本 | 与主营业务收入相对应的成本 | 主营业务成本、营业成本-主营 |
| 其他业务成本 | 与其他业务收入相对应的成本 | 其他业务成本、营业成本-其他 |
| **营业成本合计** | 主营业务成本 + 其他业务成本 | — |

> 同上，若只有"营业成本"一个总账科目则直接取值。

### 2.3 税金及附加

| 包含内容 | 说明 | 常见科目别名 |
|---------|------|-------------|
| 消费税 | 生产/委托加工/进口应税消费品缴纳的消费税 | 税金及附加-消费税 |
| 城市维护建设税 | 以实际缴纳增值税/消费税为基数计算的城建税 | 税金及附加-城建税 |
| 教育费附加 | 按缴纳增值税/消费税的一定比例计算的教育附加费 | 税金及附加-教育费附加 |
| 土地增值税 | 转让国有土地使用权等缴纳的土地增值税 | 税金及附加-土地增值税 |
| 印花税 | 书立/领受应税凭证缴纳的印花税 | 税金及附加-印花税 |
| 房产税 | 拥有或承典房产缴纳的房产税 | 税金及附加-房产税 |
| 车船税 | 拥有/使用的车辆/船舶缴纳的车船税 | 税金及附加-车船税 |
| 城镇土地使用税 | 使用城镇土地缴纳的土地使用税 | 税金及附加-土地使用税 |
| **税金及附加合计** | 以上各项之和 | 税金及附加、营业税金及附加 |

### 2.4 期间费用

| 项目 | 取数说明 | 常见科目别名 |
|------|---------|-------------|
| 销售费用 | 销售过程中发生的各项费用（运输费、广告费、销售佣金等） | 销售费用、营业费用 |
| 管理费用 | 组织管理生产经营发生的费用（行政人员薪酬、办公费、折旧等） | 管理费用 |
| 研发费用 | 研究开发活动中费用化的支出 | 研发费用、研发支出-费用化 |
| 财务费用 | 筹集资金发生的费用（利息支出-收入+手续费+汇兑损益等） | 财务费用 |
| **期间费用合计** | 销售费用 + 管理费用 + 研发费用 + 财务费用 | — |

> 注意：财务费用中的利息收入通常以负数体现（冲减费用），取数时注意符号方向。

### 2.5 其他收益

| 项目 | 取数说明 | 方向 |
|------|---------|------|
| 投资收益 | 对外投资取得的收益/损失（贷方余额为正收益） | 收益类（正数为增加利润） |
| 公允价值变动收益 | 交易性金融资产等的公允价值变动净额 | 收益类 |
| 资产处置收益 | 出售固定资产/无形资产等产生的处置利得/损失 | 收益类 |
| 信用减值损失 | 各项金融资产计提的减值损失（正数为损失，减少利润） | 损失类（正数为减少利润） |
| 资产减值损失 | 非金融资产（存货/固定资产等）计提的减值损失 | 损失类 |
| 资产处置报废损失 | 正常报废/毁损等非出售情形的处置损失 | 损失类 |
| 其他收益 | 与日常活动相关、除上述之外的政府补助等 | 收益类 |
| **其他收益净额** | ∑以上各项（收益类加正，损失类加负） | — |

### 2.6 利得与损失（营业外）

| 项目 | 取数说明 | 方向 |
|------|---------|------|
| 营业外收入 | 与日常活动无直接关系的各项利得（捐赠收入、盘盈利得、罚没收入等） | 增加利润 |
| 营业外支出 | 与日常活动无直接关系的各项损失（捐赠支出、盘亏损失、罚款支出等） | 减少利润 |
| **营业外收支净额** | 营业外收入 - 营业外支出 | — |

### 2.7 所得税费用

| 项目 | 取数说明 | 备注 |
|------|---------|------|
| 当期所得税 | 本期应交纳的企业所得税（应纳税所得额 × 适用税率） | 通常为利润总额 × 25%（小型微利企业适用优惠税率） |
| 递延所得税 | 递延所得税资产/负债的本期变动额 | = (期末递延所得税负债 - 期初) - (期末递延所得税资产 - 期初) |
| **所得税费用合计** | 当期所得税 + 递延所得税 | — |

---

## 三、核心计算公式（多步式）

```
第一步：营业毛利
  营业收入合计 - 营业成本合计 = 营业毛利

第二步：营业利润
  营业毛利
  - 税金及附加合计
  - 期间费用合计（销售费用 + 管理费用 + 研发费用 + 财务费用）
  + 其他收益净额
  = 营业利润

第三步：利润总额
  营业利润
  + 营业外收支净额（营业外收入 - 营业外支出）
  + 其他影响（如汇兑差额等）
  = 利润总额

第四步：净利润
  利润总额 - 所得税费用合计 = 净利润
```

## 四、输出格式

### 标准多步式格式

| 项目 | 本期金额 | 上期金额（可选） |
|------|---------|-----------------|
| 一、营业收入 | xxx.xx | — |
| 减：营业成本 | xxx.xx | — |
| 二、营业毛利 | xxx.xx | — |
| 减：税金及附加 | xxx.xx | — |
| 销售费用 | xxx.xx | — |
| 管理费用 | xxx.xx | — |
| 研发费用 | xxx.xx | — |
| 财务费用 | xxx.xx | — |
| 加：其他收益 | xxx.xx | — |
| 三、营业利润 | xxx.xx | — |
| 加：营业外收入 | xxx.xx | — |
| 减：营业外支出 | xxx.xx | — |
| 四、利润总额 | xxx.xx | — |
| 减：所得税费用 | xxx.xx | — |
| 五、净利润 | xxx.xx | — |

> 若用户要求精简版（仅保留一级项目），则只输出以下行：营业收入、营业成本、税金及附加、销售费用、管理费用、研发费用、财务费用、营业利润、营业外收入、营业外支出、利润总额、所得税费用、净利润。
