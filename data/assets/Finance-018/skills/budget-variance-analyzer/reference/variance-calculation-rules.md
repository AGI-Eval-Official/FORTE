# 差异计算规则

> 本文件定义 budget_variance_analyzer SKILL 在执行预算差异核算时必须遵守的计算公式、标记规则与筛选阈值。
> 所有计算必须以口径对齐后的源数据为唯一依据，禁止虚构或估算任何数值。

---

## 一、核心计算公式

### 1.1 基础差异计算

```
差异金额 = 实际支出 - 预算额度
差异率   = (差异金额 / 预算额度) × 100%
```

> 注意：预算额度为 0 时，差异率无法计算，在 notes 中标注"[科目名] 预算额度为零，差异率不适用"，差异率字段填写"N/A"。

### 1.2 汇总层计算

```
总预算额度   = Σ 各科目预算额度
总实际支出   = Σ 各科目实际支出
总差异金额   = 总实际支出 - 总预算额度
总差异率     = (总差异金额 / 总预算额度) × 100%
```

### 1.3 校验规则

```
校验项：各行差异金额之和 = 总差异金额
允许误差：浮点误差 ≤ 0.01（万元）
不通过时：记录到 notes，标注"差异金额汇总校验未通过，差值为 {X} 万元，请核查"
```

---

## 二、差异标记规则

| 差异率范围 | 标记 | 含义 |
|------------|------|------|
| 差异率 > 0% | 超支 | 实际支出超过预算 |
| 差异率 < 0% | 节约 | 实际支出低于预算 |
| 差异率 = 0% | 持平 | 实际支出等于预算 |

---

## 三、重点科目筛选规则

### 3.1 默认筛选阈值

```
重点科目触发条件：|差异率| ≥ 10%
即：差异率 ≥ 10%（超支重点科目）或 差异率 ≤ -10%（节约重点科目）
```

> 用户可通过参数 `variance_threshold` 覆盖默认阈值（如指定 5% 或 20%）。

### 3.2 特殊情形处理

- 预算额度极小（< 1 万元）但差异金额绝对值较大时：即使差异率超过阈值，在 notes 中标注"[科目名] 预算基数较小，差异率参考意义有限，差异金额为 {X} 万元"
- 差异率为 N/A 的科目：不参与重点科目筛选，单独列出

---

## 四、数值格式规范

| 类型 | 格式规范 |
|------|----------|
| 金额（计算过程） | 始终使用用户指定单位，保留 4 位小数 |
| 金额（展示） | 保留 2 位小数，附单位（如 `12.50 万元`） |
| 差异率（展示） | 百分比格式，保留 2 位小数（如 `15.23%`、`-8.50%`） |

---

## 五、用户可配置参数汇总

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `variance_threshold` | float | `10.0` | 重点科目筛选阈值（差异率绝对值，单位 %） |
| `amount_unit` | str | `"万元"` | 金额展示单位，支持"元"/"万元"/"百万元" |
| `include_zero_variance` | bool | `false` | 是否在汇总表中展示差异率为 0 的科目 |
