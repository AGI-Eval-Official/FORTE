# 预算差异分析报告模板

> 本文件定义 budget_variance_analyzer SKILL 各类输出的标准结构与格式规范。
> **格式优先级：用户明确指定输出格式时（如指定输出为 Excel、Word、纯文本等），以用户指定格式为准；未指定输出格式时，默认按照本模板输出。**
> 所有输出必须严格按照本模板填充，不得随意调整模块顺序或删减必填项。

---

## 一、执行摘要模板（必输出）

```
## 执行摘要

已完成任务：{已执行的任务 ID 及名称列表}
分析周期：{实际数据起止时间，如"2026年1月—3月"}
数据单位：{金额单位，如"万元"}
数据来源：{预算表文件名} / {支出明细表文件名} / {营收数据表文件名（若有）}
重点科目数：{超过阈值的科目总数}（超支 {X} 个 / 节约 {X} 个）
```

---

## 二、预算执行汇总表模板（O1）

### 2.1 明细表

表头顺序固定，不得调整：

| 业务线 | 费用科目 | 预算额度（{单位}） | 实际支出（{单位}） | 差异金额（{单位}） | 差异率 | 差异标记 |
|--------|----------|-------------------|-------------------|-------------------|--------|----------|
| {业务线名称} | {科目名称} | {数值} | {数值} | {数值} | {XX.XX%} | 超支/节约/持平 |

> 表格内容按业务线分组排列，同一业务线内按差异率绝对值降序排列。

### 2.2 汇总统计行

紧接明细表之后输出：

```
## 汇总统计

| 统计项 | 数值 |
|--------|------|
| 预算总额 | {XX.XX} {单位} |
| 实际支出总额 | {XX.XX} {单位} |
| 总差异金额 | {XX.XX} {单位} |
| 总差异率 | {XX.XX%} |
| 超支科目数 | {X} 个 |
| 节约科目数 | {X} 个 |
| 重点科目数（|差异率| ≥ {阈值}%） | {X} 个 |
```

---

## 三、重点科目差异分析模板（O2）

每个重点科目单独输出一个分析块，格式如下：

```
### {业务线}—{费用科目}（{超支/节约} {差异率}）

**基础数据**
- 预算额度：{XX.XX} {单位}
- 实际支出：{XX.XX} {单位}
- 差异金额：{XX.XX} {单位}
- 差异率：{XX.XX%}
- 季度营收：{XX.XX} {单位}（若无营收数据，此行省略）

**归因结论**
- 原因类别：{类别名称}
- 具体情形：{一句话描述}
- 分析依据：{可观测证据}
```

> 超支科目在前，节约科目在后；同类内部按差异率绝对值降序排列。

---

## 四、优化建议模板（O3）

每个重点科目单独输出一个建议块：

**超支科目建议格式：**

```
### {费用科目}超支优化建议（{业务线}）

**问题概述**：差异率 {XX.XX%}，超支 {XX.XX} {单位}，原因类别：{类别名称}

**建议措施**
1. {针对该科目的具体管控措施}
2. {审批流程或预算管控节点优化建议}
3. {下期预算调整方向，如"建议下期预算调整为 XX {单位}"}
```

**节约科目建议格式：**

```
### {费用科目}节约情况说明（{业务线}）

**情况概述**：差异率 {XX.XX%}，节约 {XX.XX} {单位}，原因类别：{类别名称}

**建议措施**
1. {节约预算的调配建议，如"建议将节约预算调配至{超支科目}"}
2. {成本控制经验固化建议}
3. {下期预算编制参考，如"建议下期预算按实际支出水平调整为 XX {单位}"}
```
