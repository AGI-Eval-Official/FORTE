# 品牌传播类活动规则 (brand_comm)

> 场景标识：`brand_comm`
> 适用场景：线上（品牌宣传、联名合作）；线下（发布会、展会、沙龙、快闪体验店）

---

## 一、行业评分标准

### 1.1 评分维度与权重

| 评分项 | 权重 | 评分细则 | 满分 |
|-------|------|---------|------|
| **品牌一致性** | 20% | 规则是否与品牌定位、调性、VI规范保持一致，品牌露出是否统一 | 20 |
| **合作方权益清晰度** | 15% | 联名/合作活动中各方的权益边界是否明确，有无冲突或遗漏 | 15 |
| **PR节奏合理性** | 15% | 预热→爆发→长尾的传播节奏是否合理，关键时间节点是否有对应规则 | 15 |
| **现场流程完备性** | 15% | 线下环节的动线、签到、互动、撤离等规则是否完整可执行 | 15 |
| **嘉宾/媒体接待规范** | 10% | VIP/媒体/KOL的邀请、接待、权益、应急处理是否有专门规则 | 10 |
| **内容审核机制** | 10% | UGC/媒体产出/KOL内容的审核标准、流程、时效是否明确 | 10 |
| **舆情风控覆盖** | 10% | 负面舆情的监测、分级响应、话术预案是否到位 | 10 |
| **合规性** | 5% | 广告法、肖像权、场地安全、知识产权等法律风险点是否覆盖 | 5 |

### 1.2 评分等级

| 等级 | 分数范围 | 含义 | 后续动作 |
|-----|---------|------|---------|
| S | 90-100 | 可直接执行，无需额外补充 | 输出最终文档 |
| A | 80-89 | 整体完善，少量细节需确认 | 与用户确认2-3个细节后定稿 |
| B | 70-79 | 框架完整，部分规则需补充 | 标注缺失项，请用户补全后修订 |
| C | 60-69 | 存在明显缺失，需要较大修订 | 列出问题清单，建议重写薄弱模块 |
| D | <60 | 不合格，需重新设计 | 给出具体改进方向，建议重新走SOP流程 |

### 1.3 扣分速查表

| 问题类型 | 扣分 | 说明 |
|---------|------|------|
| 品牌调性违反 | -5~-10 | 规则中出现与品牌调性不符的表述或机制 |
| 合作方权益冲突 | -5 | 两方及以上合作方权益存在重叠或矛盾 |
| PR节点缺失 | -3 | 关键传播节点（如发布会当天）无对应执行规则 |
| 现场安全漏洞 | -5~直接D | 线下活动缺少安全/消防/应急预案 |
| 媒体/VIP接待缺位 | -3 | 无专门的嘉宾接待流程或权益说明 |
| 内容审核空白 | -4 | 无任何UGC/媒体/KOL内容审核规则 |
| 舆情预案缺失 | -5 | 无负面舆情应对策略或话术 |
| 合规风险点未覆盖 | -3~直接D | 涉及肖像权/广告法/知识产权但无对应条款 |

## 二、专属违禁词库

### 2.1 品牌表述违禁词

| 违禁词 | 原因 | 替代建议 |
|-------|------|---------|
| 行业第一 / 领导者 | 无法证实，涉嫌违反《广告法》第九条 | "深受用户喜爱""行业领先品牌" |
| 最 / 顶级 / 极致 | 绝对化用语，违反《广告法》 | "优秀""高品质""出色" |
| 100%有效 / 保证成功 | 承诺过当，涉嫌虚假宣传 | "助力提升""有望达成" |
| 国家级 / 官方指定 | 未经授权不得使用 | 具体授权方名称或去掉前缀 |
| 世界级 / 国际大牌 | 无法证实且可能误导 | "具有国际影响力""广受认可" |

### 2.2 合作方/联名表述违禁词

| 违禁词 | 原因 | 替代建议 |
|-------|------|---------|
| 独家合作伙伴（非独家时） | 虚假宣传，违约风险 | "战略合作伙伴""联合呈现" |
| 官方授权（未获授权时） | 侵权风险 | "官方支持""品牌推荐" |
| XX品牌倾力推荐（付费合作） | 未标明广告，违反《广告法》 | 明确标注"广告""赞助""合作" |
| 限量XX份（实际不限） | 虚假营销 | 如实标注数量或使用"数量有限" |

### 2.3 线下活动专属违禁词

| 违禁词 | 原因 | 替代建议 |
|-------|------|---------|
| 必须到场 / 强制参加 | 强制性表述，用户体验差 | "诚邀出席""期待您的参与" |
| 无限畅饮 / 随便吃 | 可能引发过度消耗和安全问题 | "提供餐饮""品鉴区开放" |
| 现场送豪礼（无明细） | 信息不完整，引发纠纷 | 明确礼品内容和获取条件 |
| 名额有限（实际大量空余） | 制造虚假紧迫感 | 如实说明名额情况或"预约制" |

### 2.4 KOL/媒体表述违禁词

| 违禁词 | 原因 | 替代建议 |
|-------|------|---------|
| 网红 / 大V（正式文档中） | 不够专业 | "内容创作者""KOL""达人" |
| 疯传 / 刷屏 | 夸大效果预期 | "广泛传播""获得关注" |
| 粉丝量XXX万（需核实） | 数据可能失真 | "拥有广泛受众基础"或核实后使用精确数据 |
| 独家爆料 / 内幕消息 | 可能涉及商业机密或法律风险 | "深度解读""独家视角" |

### 2.5 敏感话题规避

以下话题在品牌传播类活动中应主动规避或经法务/PR审批后方可使用：

| 敏感领域 | 规避原因 | 处理方式 |
|---------|---------|---------|
| 政治/宗教相关 | 品牌中立性要求 | 完全避免 |
| 竞品对比（贬低性） | 不正当竞争风险 | 仅做客观参数对比，不做主观评价 |
| 数据来源不明 | 虚假宣传风险 | 所有引用数据必须标注出处 |
| 用户隐私收集声明模糊 | 个人信息保护法规要求 | 必须明确告知收集目的、方式、范围 |
| 环保/可持续宣称 | "漂绿"监管趋严 | 有第三方认证方可使用相关表述 |

## 三、输出文档结构参考

### 3.1 文档章节框架

基本信息 → 品牌规范（调性VI/露出要求） → 合作方规则（权责划分/权益分配/联合宣发） → 传播节奏规划（预热/爆发/长尾） → 现场执行规则（场地动线/签到入场/流程环节/安全应急） → 嘉宾/媒体接待（邀请名单/接待流程/专访安排） → 内容与审核（UGC规范/KOL要求/审核流程） → 风控与异常处理（舆情监测/异常预案/合规审查） → 附件（FAQ/话术/物料清单）
