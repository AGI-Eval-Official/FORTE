# 行业关键词维度映射

本文件用于定义不同业务场景下关键词提取的推荐维度。当 Skill 接收到 `theme_dimensions` 参数时，可参考以下映射自动补全或校验。

## 电商 / 消费品
- 产品属性：质量、外观、尺寸、重量、材质、颜色
- 价格：性价比、折扣、贵、便宜、划算
- 物流：快递速度、包装、配送服务
- 售后：退换货、客服态度、保修
- 使用体验：操作便捷、效果、耐用性

## 餐饮 / 食品
- 口味：咸淡、甜度、辣度、新鲜度、分量
- 服务：上菜速度、服务员态度、环境
- 卫生：清洁度、食材新鲜
- 价格：人均、团购、优惠
- 复购意愿：还会再来、推荐朋友

## 互联网产品 / App
- 功能：易用性、稳定性、加载速度、界面设计
- 性能：卡顿、闪退、耗电、流量消耗
- 客服：响应速度、解决问题能力
- 价值：有用、效率提升、省钱

## 汽车 / 出行
- 动力：加速、油耗/电耗、发动机/电机
- 操控：转向、底盘、刹车
- 空间：乘坐空间、后备箱
- 内饰：材质、设计、异味
- 智能：车机、辅助驾驶、OTA
- 售后：保养、维修、服务网点

## 旅游 / 酒店
- 住宿：卫生、床品、隔音、热水、WiFi
- 服务：前台态度、早餐、接送
- 位置：交通便利、周边配套
- 价格：性价比、隐藏费用

## 教育 / 课程
- 内容：知识深度、实用性、更新频率
- 讲师：专业度、讲解清晰、互动
- 平台：播放流畅、功能易用
- 效果：学完能用、提分、考证通过

## 通用（无行业指定）
- 默认维度：满意点、不满意点、质量、服务、价格、体验、问题、建议
