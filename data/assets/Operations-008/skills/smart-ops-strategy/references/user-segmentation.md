# 用户分层框架与策略方向

## 通用用户生命周期分层

### 一、新用户（注册/首购 ≤ 30天）

**核心目标**：完成首单转化 / 建立使用习惯 / 提升留存

**关键行为信号**：
- 注册未购买（潜在新用户）
- 完成首单但未复购
- 浏览多次但未下单

**策略方向**：

1. **新手引导激励**：首单立减/免运费/新人专属礼包，降低首次决策门槛。重点在于让用户"先用起来"，而不是追求高客单价。

2. **品类种草**：根据用户浏览行为推荐最可能感兴趣的品类，用内容（评测、榜单）而非广告建立信任。

3. **快速建立复购习惯**：首单完成后48小时内触达，推荐关联品类或"下次购买"优惠，趁热打铁。

4. **降低流失风险**：注册后3天未有任何行为，发送激活提醒；7天未购买，发送限时优惠。

**触达建议**：
- 渠道：App推送 > 短信（新用户对推送接受度较高）
- 频次：前7天可适当高频（每2-3天一次），之后降频
- 文案方向：突出"专属"、"限时"、"新人特权"

---

### 二、成长用户（购买2-5次，注册30-180天）

**核心目标**：提升购买频次 / 扩大品类渗透 / 培养品牌偏好

**关键行为信号**：
- 有固定购买品类但未跨品类
- 客单价稳定但未提升
- 偶尔使用优惠券

**策略方向**：

1. **品类扩展**：基于已购品类推荐相关品类（如买了咖啡豆，推荐咖啡机/滤纸），用"搭配购买"逻辑提升品类渗透。

2. **频次激励**：设置"连续购买奖励"或"月度任务"，让用户形成周期性购买习惯。

3. **会员体系引导**：引导加入会员/积分体系，用长期权益绑定用户，提升迁移成本。

4. **社交裂变**：成长期用户对平台有一定信任，适合推荐有礼活动，既能拉新又能增强用户归属感。

**触达建议**：
- 渠道：App推送 + 个性化首页推荐
- 频次：每周1-2次，避免打扰
- 文案方向：突出"专属推荐"、"为你精选"

---

### 三、高价值用户（高频高消费，通常为RFM高分用户）

**核心目标**：维系关系 / 防止流失 / 提升客单价 / 口碑传播

**关键行为信号**：
- 近期有购买（Recency低）
- 购买频次高（Frequency高）
- 消费金额大（Monetary高）
- 可能已是会员/VIP

**策略方向**：

1. **专属权益维系**：提供差异化的VIP服务（专属客服、优先发货、专属折扣），让高价值用户感受到"被重视"，而不只是发优惠券。

2. **提前预警流失**：高价值用户一旦流失损失极大。设置预警机制：如果高价值用户超过其平均复购周期的1.5倍未购买，立即触发关怀策略。

3. **高端品类/服务推荐**：推荐更高客单价的商品或增值服务，用"升级体验"而非"打折"来提升消费。

4. **KOC转化**：邀请高价值用户参与内测、评测、社群，转化为口碑传播者。

**触达建议**：
- 渠道：专属客服 > 个性化推送 > 短信（高价值用户对打扰更敏感）
- 频次：低频精准，每月2-3次即可
- 文案方向：突出"专属"、"尊享"、"优先"，避免大众化促销话术

---

### 四、沉睡用户（有购买历史，但近期无活跃）

**核心目标**：重新激活 / 了解流失原因 / 分级召回

**判断标准**（根据业务复购周期调整）：
- 轻度沉睡：超过平均复购周期1-2倍未购买
- 中度沉睡：超过平均复购周期2-3倍未购买
- 深度沉睡：超过平均复购周期3倍以上未购买

**策略方向**：

1. **分级召回，成本递增**：
   - 轻度沉睡：个性化推荐 + 小额优惠（5-10元券）
   - 中度沉睡：较大力度优惠（满减/折扣）+ 情感化文案
   - 深度沉睡：高价值用户用大额优惠，低价值用户考虑放弃或低成本触达

2. **找回流失原因**：通过问卷或客服了解流失原因，针对性解决（如物流差→承诺改善，价格贵→专属折扣）。

3. **场景化唤醒**：结合节假日、用户历史购买周期（如去年同期）进行唤醒，"去年这时候你买了XX，今年还需要吗？"

4. **降低回购门槛**：提供免运费、小额试用装等低门槛回购选项，让用户"先回来再说"。

**触达建议**：
- 渠道：短信（沉睡用户可能已关闭App推送）> 邮件 > App推送
- 频次：召回序列通常3-5次，间隔3-7天，无响应则停止
- 文案方向：情感化（"好久不见"）或利益驱动（"专属回归礼"），避免过于商业化

---

### 五、流失用户（长期无活跃，基本判定流失）

**核心目标**：低成本尝试召回 / 了解流失原因 / 控制召回成本

**策略方向**：

1. **高价值流失用户**：值得投入较高成本召回，可提供大额优惠或专属服务。

2. **普通流失用户**：低成本触达（短信/邮件），不值得投入大额优惠。

3. **放弃策略**：对于长期无响应的低价值流失用户，停止触达，节省资源。

4. **流失分析**：定期分析流失用户特征，反哺产品和运营改进。

## 按消费行为的细分分层

### 价格敏感型用户
**特征**：优惠券使用率高、参与秒杀/拼团多、客单价低
**策略**：满减活动、限时折扣、拼团邀请；避免推送原价商品

### 品质导向型用户
**特征**：客单价高、购买高端品类、很少使用优惠券
**策略**：新品首发、品质内容推荐、会员专属服务；避免频繁发券

### 冲动消费型用户
**特征**：参与秒杀/预售多、购买频次高但不规律、加购后转化率高
**策略**：限时限量活动、倒计时提醒、加购未付款提醒

### 理性研究型用户
**特征**：浏览时间长、收藏多但购买少、查看评价多
**策略**：详细的商品评测内容、用户口碑展示、"犹豫期"专属优惠

### 礼品购买型用户
**特征**：节假日前后购买集中、购买礼品包装选项多
**策略**：节日前提前触达、礼品推荐、包装/卡片服务

## 按地域/时段的分层补充

### 下沉市场用户
**特征**：价格敏感、社交裂变参与度高、使用时段集中在晚间
**策略**：拼团/砍价活动、社群运营、晚间推送

### 一线城市用户
**特征**：时间敏感、品质要求高、即时配送需求强
**策略**：极速配送权益、高端品类推荐、碎片化时间触达（通勤时段）
