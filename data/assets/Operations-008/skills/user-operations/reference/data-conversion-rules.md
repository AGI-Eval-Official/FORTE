# 转化：复购分析 · 价值评估

## 定位

围绕「转化」目标，通过复购分析和用户价值评估**衡量用户的转化深度和商业价值**，识别高价值用户群体并理解重复行为模式，为提升用户终身价值（LTV）和优化转化策略提供数据依据。

---

## 1. 复购分析

针对有交易/转化行为的用户，分析重复行为模式：

| 分析项 | 内容 | 默认规则 |
|--------|------|---------|
| 复购率 | 有 ≥2 次目标行为的用户数 / 有过至少 1 次目标行为的用户数 | 目标事件默认取订单/下单类行为，用户可指定；**百分比保留1位小数** |
| 复购周期 | 相邻两次目标行为的平均时间间隔 | 计算均值和中位数；**天数保留1位小数（如45.3天）** |
| 首次复购时间 | 从首次行为到第二次行为的平均天数 | 按 Cohort 分组展示 |
| 复购次数分布 | 用户复购次数的分布（1 次 / 2 次 / 3+ 次 ...） | 输出频数和占比 |

> 复购分析的「目标行为」**用户可自定义**（如定义为"加购""会员开通"等非订单行为）。用户指定时严格按用户定义执行。

### 1.1 复购与促活的联动

复购分析与「促活」（Data_activation_rules.md）中的 RFM 分析高度关联：
- **RFM 的 F（频次）维度**反映的是行为频次，是复购分析的基础指标
- **RFM 的 M（金额）维度**反映的是价值贡献，与复购金额直接相关
- 建议在执行 RFM 分析后，对高 F 高 M 群体进一步做复购细分分析

### 1.2 复购与留存的联动

复购是留存的高级形式——用户不仅"留下来"，还"反复转化"。建议结合留存分析（Data_retention_rules.md）一起看：
- 高留存 + 低复购 → 用户活跃但未转化为付费/交易
- 低留存 + 高复购 → 用户生命周期短但转化密度高（需关注流失原因）

## 2. 用户价值评估

### 2.1 价值维度

当数据中存在金额/价值型字段时，进行以下评估：

| 评估项 | 说明 | 输出 |
|--------|------|------|
| 用户价值分布 | 按累计金额/价值的分档分布 | 各档人数及占比（高/中/低） |
| 价值集中度 | TOP N 用户贡献的占比（如 TOP 10% 用户贡献了多少收入） | 累计占比曲线或帕累托分析 |
| 单用户平均价值 | 总价值 / 用户数 | 均值、中位数 |
| 价值趋势 | 不同时期新用户的价值对比 | 按注册时间/获客批次分组展示 |

### 2.2 无金额字段时的降级方案

当数据中没有金额/价值字段时：

- 使用**行为频次**作为价值代理指标（如浏览次数、登录次数等）
- 在输出中明确标注："以[具体行为]频次作为价值代理指标"
- 或跳过本模块，仅执行其他非价值类分析
