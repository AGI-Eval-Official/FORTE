# 参考文档：数据看板工具库

## 项目概述

`dashboard-toolkit` 是一个数据看板的前端工具库，提供指标计算、数据转换、筛选过滤等纯工具函数。

## 技术栈

- Vue 3 生态
- Vitest 测试框架（基于 Vite）
- ESM 模块系统

## 业务术语

| 术语 | 含义 |
|------|------|
| 增长率（Growth Rate） | 衡量指标在两个时期间的变化幅度，计算公式为 (新值 - 旧值) / 旧值 |
| 转化率（Conversion Rate） | 目标完成数占总数的比例 |
| 日期范围筛选 | 按起止日期过滤数据，起止日期均应包含在结果中（闭区间） |
| 数据排序 | 对数据集按指定字段排序，应返回新数组，不改变原始数据 |

## 注意事项
- 运行测试：`npx vitest run`
