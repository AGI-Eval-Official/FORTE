# 输出文档排版规范

本规则定义生成的 .docx 考试文档的完整排版标准，包括模板解析、页面设置、字体、章节结构、各题型排版格式和答案速查表。

当用户提供了排版模板时，模板样式**优先于**本规则中的默认值；模板未覆盖的排版项，仍使用本规则的默认值。

---

## 1. 模板解析

当用户提供 .docx 排版模板时，按以下规则提取模板样式并应用：

### 1.1 提取项

| 提取项 | 从模板中提取的内容 | 提取方式 |
|--------|-------------------|----------|
| 页面设置 | 上/下/左/右边距 | 读取模板的 Section Properties |
| 标题字体 | 文档标题的字体、字号、颜色 | 读取模板中第一个 Heading 级别样式的属性 |
| 正文字体 | 正文的字体、字号、颜色 | 读取模板的默认段落样式 (Normal) |
| 行间距 | 正文行间距、段后间距 | 读取模板的段落格式属性 |
| 表格样式 | 表头背景色、字体、字号 | 读取模板中表格的样式定义 |
| 编号格式 | 题号格式、章节编号格式 | 分析模板中已有列表的编号样式 |

### 1.2 提取规则

1. **逐项提取，可用即用**：对每个提取项独立尝试提取，某项提取失败不影响其他项的应用。
2. **模板优先，默认兜底**：模板中能提取到的样式项，覆盖本规则 §2-§3 的默认值；提取不到的项，使用默认值。
3. **忽略模板内容，只取样式**：模板中的文字内容不参与出题，仅提取排版样式属性。
4. **编号格式推断**：若模板中包含带编号的列表，分析其编号样式（如 `1.` / `（1）` / `一、` 等）并应用到对应场景；无法推断时使用默认编号格式。

### 1.3 模板异常处理

| 异常情况 | 处理方式 |
|---------|----------|
| 模板文件不存在 | 记录模板缺失，使用默认排版继续生成 |
| 模板文件格式损坏 | 记录模板损坏，使用默认排版继续生成 |
| 模板为空或无样式定义 | 记录模板样式缺失，使用默认排版继续生成 |

## 2. 页面设置

> 提供模板时，从模板提取；未提供时使用以下默认值。

| 项目 | 规格 |
|------|------|
| 上边距 | 2.5cm |
| 下边距 | 2.5cm |
| 左边距 | 2.5cm |
| 右边距 | 2.5cm |

---

## 3. 字体规范

> 提供模板时，从模板提取标题和正文字体属性；未提供时使用以下默认值。

| 用途 | 首选字体 | 回退字体 | 字号 | 颜色 |
|------|---------|---------|------|------|
| 文档标题 | 微软雅黑 (Microsoft YaHei) | 系统默认无衬线字体 | Heading 0 级别 | #1F497D（深蓝） |
| 章节标题 | 微软雅黑 (Microsoft YaHei) | 系统默认无衬线字体 | Heading 2 级别 | #1F497D（深蓝） |
| 题目正文 | 宋体 (SimSun) | 系统默认衬线字体 | 11pt | 黑色 |
| 答案速查表表头 | 微软雅黑 (Microsoft YaHei) | 系统默认无衬线字体 | 10pt | 白色（#FFFFFF） |
| 答案速查表数据 | 宋体 (SimSun) | 系统默认衬线字体 | 10pt | 黑色 |

> **字体回退**：当系统无微软雅黑或宋体时，使用系统默认字体，不得报错或中断生成。

---

## 4. 行间距与段间距

> 提供模板时，从模板提取间距属性；未提供时使用以下默认值。

| 项目 | 规格 |
|------|------|
| 正文行间距 | 20pt |
| 段后间距 | 6pt |
| 题目与选项/作答区间距 | 2pt |
| 选项与选项间距 | 1pt |
| 题目之间间距 | 12pt |
| 说明文字段后间距 | 10pt |

---

## 5. 文档结构

### 5.1 总体结构

文档结构根据启用的题型**动态生成**。题型顺序固定为：选择题 → 简答题 → 判断题 → 填空题，未启用的题型跳过。

示例（选择题 + 判断题 + 填空题，无简答题）：

```
[文档标题]
[分隔线]
一、选择题
[选择题说明]
[题目1-10]
二、判断题
[判断题说明]
[题目11-13]
三、填空题
[填空题说明]
[题目14-15]
答案速查表
```

### 5.2 章节编号规则

- 默认按"一、二、三、四、…"中文序号递进；用户指定或模板推断出其他格式时，以用户/模板为准。
- 仅对实际启用的题型编号，跳过未启用的题型。
- 章节编号与题型顺序绑定：第一个启用题型为第 1 项，第二个为第 2 项，以此类推。

常见章节编号格式：

| 格式 | 示例 |
|------|------|
| 中文序号 | 一、选择题 / 二、简答题 |
| 阿拉伯数字+点 | 1. 选择题 / 2. 简答题 |
| 带括号数字 | （一）选择题 / （二）简答题 |

### 5.3 文档标题

- 居中对齐。
- 使用 Heading 0 级别。
- 内容为 `exam_title` 字段值。
- 颜色 #1F497D。

### 5.4 分隔线

- 标题下方插入水平分隔线；若模板中存在分隔线样式则以模板为准，否则插入一条横跨页面的细实线。

---

## 6. 选择题格式

> 题号格式默认为阿拉伯数字+英文句点（如 `1.`），用户指定或模板推断出其他格式时，以用户/模板为准。

### 6.1 章节标题

- 文本：`{序号}、选择题`（如 `一、选择题`）
- 级别：Heading 2
- 颜色：#1F497D

### 6.2 说明文字

- 文本：`（共 X 题，每题含 A、B、C、D 四个选项，只有一个正确答案）`
- 字号：10pt
- 段后间距：10pt

### 6.3 题目排版

每道选择题的排版顺序：

1. **题号 + 题目**：`{id}. {question}`，加粗，段后间距 2pt
2. **选项 A**：`    A. {选项内容}`，缩进，段后间距 1pt
3. **选项 B**：`    B. {选项内容}`，缩进，段后间距 1pt
4. **选项 C**：`    C. {选项内容}`，缩进，段后间距 1pt
5. **选项 D**：`    D. {选项内容}`，缩进，段后间距 1pt
6. **空白行**：段后间距 12pt（与下一题分隔）

> **关键要求**：题目中**不显示答案和解析**，答案仅在速查表中呈现。

---

## 7. 简答题格式

### 7.1 章节标题

- 文本：`{序号}、简答题`（如 `二、简答题`）
- 级别：Heading 2
- 颜色：#1F497D

### 7.2 说明文字

- 文本：`（共 X 题，请结合培训资料作答）`
- 字号：10pt
- 段后间距：10pt

### 7.3 题目排版

每道简答题的排版顺序：

1. **题号 + 题目**：`{id}. {question}`，加粗，段后间距 2pt
2. **作答标识**：另起一行，`答:`，段后间距 12pt

> **关键要求**：题目中**不显示参考答案**，参考答案仅在速查表中呈现。题干中必须另起一行包含"答:"。

---

## 8. 判断题格式

### 8.1 章节标题

- 文本：`{序号}、判断题`（如 `二、判断题`，视位置而定）
- 级别：Heading 2
- 颜色：#1F497D

### 8.2 说明文字

- 文本：`（共 X 题，请判断以下命题的对错）`
- 字号：10pt
- 段后间距：10pt

### 8.3 题目排版

每道判断题的排版顺序：

1. **题号 + 命题**：`{id}. {question}`，加粗，段后间距 2pt
2. **作答区**：另起一行，`（  ）`，段后间距 12pt

> **关键要求**：题目中**不显示答案和解析**，答案仅在速查表中呈现。

---

## 9. 填空题格式

### 9.1 章节标题

- 文本：`{序号}、填空题`（如 `三、填空题`，视位置而定）
- 级别：Heading 2
- 颜色：#1F497D

### 9.2 说明文字

- 文本：`（共 X 题，请在横线处填入正确内容）`
- 字号：10pt
- 段后间距：10pt

### 9.3 题目排版

每道填空题的排版顺序：

1. **题号 + 题目**：`{id}. {question}`（含 `___` 标记），加粗，段后间距 12pt

> **关键要求**：题目中**不显示答案和解析**，答案仅在速查表中呈现。

---

## 10. 答案速查表

### 10.1 位置

- 位于文档最末尾。
- 标题：`答案速查表`，级别 Heading 2，颜色 #1F497D。

### 10.2 表格结构

4列：`题型 | 题号 | 正确答案 | 解析摘要`

### 10.3 表头样式

> 提供模板时，从模板提取表格样式；未提供时使用以下默认值。

- 背景色：#1F497D（深蓝）
- 文字颜色：白色（#FFFFFF）
- 字体：微软雅黑 10pt，加粗
- 对齐：居中

### 10.4 数据行样式

- 字体：宋体 10pt
- 对齐：居中
- 解析摘要截断规则：超过40字符时，在最近的中文字/标点后截断并加 `…`

### 10.5 各题型数据行内容

| 题型 | 题号列 | 正确答案列 | 解析摘要列 |
|------|--------|-----------|-----------|
| 选择题 | `{id}` | `{answer}`（如 A） | 截断后的 `explanation` |
| 简答题 | `{id}` | `见参考答案` | 截断后的 `reference_answer` |
| 判断题 | `{id}` | `正确 (√)` 或 `错误 (×)` | 截断后的 `explanation` |
| 填空题 | `{id}` | `{blank_answer}` | 截断后的 `explanation` |

### 10.6 数据行顺序

按题型顺序排列：选择题 → 简答题 → 判断题 → 填空题，各题型内按题号升序。

---

## 11. 输出文件

- 格式：.docx
- 文件名：用户指定优先；未指定时默认为 `培训验收.docx`
- 输出路径：用户指定优先；未指定时默认为输入文件所在目录，目录不可写时自动回退到当前工作区目录

### 11.1 输出格式兜底规则

- 默认值（文件名、路径、编号、样式）仅作为示例，不覆盖用户明确要求。
- 输出相关规则冲突时按「用户输入 > `SKILL.md` 主流程约束 > 本规则默认示例」执行。
- 除文件系统不可写且无可用回退目录等阻断条件外，不进入反问，自动选择可执行输出方案。

---

## 12. 兜底策略（用户输入优先原则）

在所有排版环节中，**用户输入为第一优先级**：

| 场景 | 处理规则 |
|------|---------|
| 用户提供排版模板 | 模板中定义的页面设置、字体、间距、颜色、编号格式等样式优先于默认规则 |
| 用户指定题号格式 | 使用用户指定的编号格式（如"1、""（1）""①"等），覆盖默认的"1."格式 |
| 用户指定章节编号格式 | 使用用户指定的章节编号格式（如"第一章""Part 1"等），覆盖默认的"一、"格式 |
| 用户指定页面边距 | 使用用户指定的边距值，覆盖默认 2.5cm |
| 用户指定字体样式 | 使用用户指定的字体、字号、颜色，覆盖默认字体规范 |
| 用户指定行间距 | 使用用户指定的行间距和段间距，覆盖默认值 |
| 用户指定题型顺序 | 按用户要求的顺序排列题型章节，不按默认顺序 |
| 用户指定输出文件名 | 使用用户指定的文件名，覆盖默认"培训验收.docx" |

> **核心原则**：用户明确指定的任何排版参数（格式、样式、布局、顺序等）均以用户输入为准，本规则仅作为用户未指定时的默认示例。
