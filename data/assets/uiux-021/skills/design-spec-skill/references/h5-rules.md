# H5 规则

## 适用范围

用于移动网页、活动 H5 页面和浏览器访问场景的设计规范。

## 用户可指定参数（本平台）

用户在需求中可直接指定以下参数：
- 单位体系（`px` 或 `rem`，默认 `rem`）
- `rem` 基准字号
- 色彩（主色、功能色、中性色、状态色）
- 字体（字号梯度、字重、行高、最小正文字号）
- 间距刻度与基础步长
- 圆角与阴影层级
- 图标尺寸与线宽
- 组件尺寸与状态样式
- 输出格式（`md`/`html`/`docx`/`json`）
- 输出内容范围（章节选择、包含项、排除项）

未指定字段按“未指定时默认参数表”补齐。

## 未指定时默认参数表

### 单位与布局默认值
- 默认单位：`rem`（用户未指定时）
- 基准字号：`1rem = 16px`
- 默认间距刻度：`0.25rem` / `0.5rem` / `0.75rem` / `1rem` / `1.5rem` / `2rem`
- 最小触控尺寸：`2.75rem x 2.75rem`（约 44px）

### 色彩默认值
- 主色：`#1677FF`
- 成功色：`#16A34A`
- 警告色：`#F59E0B`
- 错误色：`#DC2626`
- 主文字色：`#1F2937`
- 次文字色：`#6B7280`
- 主背景色：`#FFFFFF`

### 字体默认值
- 正文字号建议：`0.875rem` ~ `1rem`
- 字重：`400` / `500` / `600`
- 默认行高系数：`1.5`

### 图标默认值
- 尺寸：`1rem` / `1.25rem` / `1.5rem`
- 线宽：`1.5`

### 圆角与阴影默认值
- 圆角：`0.25rem` / `0.5rem` / `0.75rem`
- 阴影：
  - `shadow-1`: `0 1px 2px 0 rgba(0,0,0,0.08)`
  - `shadow-2`: `0 4px 12px 0 rgba(0,0,0,0.12)`

### 组件默认值
至少定义：
- 按钮（默认/悬停/激活/禁用）
- 输入框（默认/聚焦/错误/禁用）
- 卡片或面板容器

## 覆盖规则

- 用户显式给出的字号、色彩、间距、圆角、阴影、组件尺寸与状态，优先级高于本文件默认值。
- 用户仅指定部分参数时，仅覆盖对应字段，其余继续使用本文件默认值。

## 交互说明

- 需要说明窄屏下的响应式处理。
- 悬停态仅在桌面 Web 访问需要时补充。

## 输出注意

- 在文档开头写明单位体系。
- 补充常见手机宽度下的约束建议。
