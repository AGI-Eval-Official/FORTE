# Android 规则

## 适用范围

当用户明确要求 Android 设计规范时使用。

## 用户可指定参数（本平台）

用户在需求中可直接指定以下参数：
- 单位体系（布局 `dp`、字号 `sp`）
- 色彩（主色、功能色、中性色、状态色）
- 字体（字号梯度、字重、行高、最小正文字号）
- 间距刻度与基础步长
- 圆角与阴影层级
- 图标尺寸与线宽
- 组件尺寸与状态样式
- 输出格式（`md`/`html`/`docx`/`json`）
- 输出内容范围（章节选择、包含项、排除项）

未指定字段按“未指定时默认参数表”补齐。

## 未指定时默认参数表

### 单位与布局默认值
- 默认单位：布局 `dp`，字号 `sp`
- 默认间距刻度：`4dp` / `8dp` / `12dp` / `16dp` / `24dp` / `32dp`
- 最小触控尺寸：`48dp x 48dp`

### 色彩默认值
- 主色：`#6750A4`
- 成功色：`#2E7D32`
- 警告色：`#ED6C02`
- 错误色：`#B3261E`
- 主文字色：`#1C1B1F`
- 次文字色：`#49454F`
- 主背景色：`#FFFBFE`

### 字体默认值
- 正文字号建议：`14sp` ~ `16sp`
- 字重：`400` / `500` / `600`
- 默认行高系数：`1.45`

### 图标默认值
- 尺寸：`18dp` / `24dp` / `32dp`
- 线宽：`1.5`

### 圆角与阴影默认值
- 圆角：`4dp` / `8dp` / `12dp`
- 阴影：
  - `shadow-1`: `0 1dp 2dp 0 rgba(0,0,0,0.10)`
  - `shadow-2`: `0 4dp 12dp 0 rgba(0,0,0,0.14)`

### 组件默认值
至少定义：
- 主按钮与次按钮状态样式
- 输入框状态样式
- 顶部应用栏与底部导航说明

## 覆盖规则

- 用户显式给出的字号、色彩、间距、圆角、阴影、组件尺寸与状态，优先级高于本文件默认值。
- 用户仅指定部分参数时，仅覆盖对应字段，其余继续使用本文件默认值。

## 交互说明

- 使用清晰的层级与阴影表达界面结构。
- 触控反馈状态要明显且一致。

## 输出注意

- 在表格中明确区分 `dp` 与 `sp`。
- token 命名保持简洁并易于研发映射。
