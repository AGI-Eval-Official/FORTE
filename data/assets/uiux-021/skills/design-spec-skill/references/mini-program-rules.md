# 小程序规则

## 适用范围

用于微信小程序或类似小程序场景的设计规范输出。

## 用户可指定参数（本平台）

用户在需求中可直接指定以下参数：
- 单位体系（默认 `rpx`）
- 色彩（主色、功能色、中性色、状态色）
- 字体（字号梯度、字重、行高、最小正文字号）
- 间距刻度与基础步长
- 圆角与阴影层级
- 图标尺寸与线宽
- 组件尺寸与状态样式
- 输出格式（`md`/`html`/`docx`/`json`）
- 输出内容范围（章节选择、包含项、排除项）

未指定字段按“未指定时默认参数表”补齐。

## 未指定时默认参数表

### 单位与布局默认值
- 默认单位：`rpx`
- 默认间距刻度：`8rpx` / `12rpx` / `16rpx` / `24rpx` / `32rpx`
- 最小触控尺寸：`88rpx x 88rpx`

### 色彩默认值
- 主色：`#1677FF`
- 成功色：`#16A34A`
- 警告色：`#F59E0B`
- 错误色：`#DC2626`
- 主文字色：`#1F2937`
- 次文字色：`#6B7280`
- 主背景色：`#FFFFFF`

### 字体默认值
- 正文字号建议：`24rpx` ~ `28rpx`
- 字重：`400` / `500` / `600`
- 默认行高系数：`1.5`

### 图标默认值
- 尺寸：`32rpx` / `40rpx` / `48rpx`
- 线宽：`1.5`

### 圆角与阴影默认值
- 圆角：`8rpx` / `12rpx` / `16rpx`
- 阴影：
  - `shadow-1`: `0 2rpx 6rpx 0 rgba(0,0,0,0.08)`
  - `shadow-2`: `0 8rpx 24rpx 0 rgba(0,0,0,0.12)`

### 组件默认值
至少定义：
- 按钮（主按钮/次按钮/禁用）
- 表单输入与选择器状态
- 列表项与卡片样式

## 覆盖规则

- 用户显式给出的字号、色彩、间距、圆角、阴影、组件尺寸与状态，优先级高于本文件默认值。
- 用户仅指定部分参数时，仅覆盖对应字段，其余继续使用本文件默认值。

## 交互说明

- 优先考虑快速浏览与单手操作。
- 信息流页面用清晰分隔提升扫描效率。

## 输出注意

- 文档中明确标注单位为 `rpx`。
- token 设计保持精简，方便复用。
