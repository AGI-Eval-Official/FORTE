# iOS 规则

## 适用范围

当用户明确要求 iOS 设计规范时使用。

## 用户可指定参数（本平台）

用户在需求中可直接指定以下参数：
- 单位体系（默认 `pt`）
- 色彩（主色、功能色、中性色、状态色）
- 字体（字号梯度、字重、行高、最小正文字号）
- 间距刻度与基础步长
- 圆角与阴影层级
- 图标尺寸与线宽
- 组件尺寸与状态样式
- 输出格式（`md`/`html`/`docx`/`json`）
- 输出内容范围（章节选择、包含项、排除项）

未指定字段按“未指定时默认参数表”补齐。

## 未指定时默认参数表

### 单位与布局默认值
- 默认单位：`pt`
- 默认间距刻度：`4pt` / `8pt` / `12pt` / `16pt` / `24pt` / `32pt`
- 最小触控尺寸：`44pt x 44pt`

### 色彩默认值
- 主色：`#0A84FF`
- 成功色：`#30D158`
- 警告色：`#FF9F0A`
- 错误色：`#FF453A`
- 主文字色：`#1C1C1E`
- 次文字色：`#636366`
- 主背景色：`#FFFFFF`

### 字体默认值
- 正文字号建议：`14pt` ~ `17pt`
- 字重：`400` / `500` / `600`
- 默认行高系数：`1.45`

### 图标默认值
- 尺寸：`16pt` / `20pt` / `24pt`
- 线宽：`1.5`

### 圆角与阴影默认值
- 圆角：`6pt` / `10pt` / `14pt`
- 阴影：
  - `shadow-1`: `0 1pt 2pt 0 rgba(0,0,0,0.08)`
  - `shadow-2`: `0 6pt 16pt 0 rgba(0,0,0,0.12)`

### 组件默认值
至少定义：
- iOS 风格主按钮与次按钮
- 输入框（默认/聚焦/错误/禁用）
- 导航栏与 Tab 栏尺寸说明

## 覆盖规则

- 用户显式给出的字号、色彩、间距、圆角、阴影、组件尺寸与状态，优先级高于本文件默认值。
- 用户仅指定部分参数时，仅覆盖对应字段，其余继续使用本文件默认值。

## 交互说明

- 贴合 iOS 交互习惯，强调清晰和克制。
- 视觉层级轻量，避免过度装饰。

## 输出注意

- 明确标注单位为 `pt`。
- 补充 iOS 组件行为要点。
