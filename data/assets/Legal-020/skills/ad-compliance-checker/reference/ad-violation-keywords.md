# 广告法常见违规词汇参考库

> 本文件为 ad-compliance-checker-v2 的通用违规词汇知识库，供审查时关键词匹配使用。
> 命中词汇需结合语境判断是否构成实质违规，不得机械匹配。

---

## 一、绝对化用语类（对应广告法第九条第三项）

| 违规词汇 | 建议替代表述 |
|----------|-------------|
| 第一、全国第一、行业第一 | 领先、知名、头部 |
| 唯一、独一无二 | 特色、专属 |
| 最好、最佳、最优 | 优质、优良、出色 |
| 最大、最强、最快 | 大型、强劲、高效 |
| 顶级、极品、至尊 | 高端、精品、臻品 |
| 独家、首创 | 特色、创新 |
| 无与伦比、无可匹敌 | 独具特色、表现卓越 |
| 史上最、有史以来最 | （删除，重新描述） |
| 终极、终极体验 | 卓越、出色 |

## 二、功效承诺类（对应广告法第四条、第十六条、第十七条）

| 违规表述 | 问题类型 |
|----------|----------|
| 100%有效、百分百见效 | 效果绝对化保证 |
| 包治、根治、彻底治愈 | 医疗效果承诺 |
| 立即见效、即刻起效 | 时间承诺 |
| 永不复发、一次解决 | 效果永久性保证 |
| 无副作用、零副作用 | 安全性绝对化承诺 |
| 临床证明、经科学验证（无来源） | 无依据的权威背书 |
| 治疗、治愈、康复（普通食品/保健品） | 普通商品宣称治疗功能 |
| 增强免疫力（无资质产品） | 保健功效违规宣称 |

---

## 三、价格误导类（对应广告法第二十八条）

| 违规表述 | 问题类型 |
|----------|----------|
| 史上最低价、历史最低 | 无依据价格对比 |
| 全网最低、全平台最低 | 虚假价格承诺 |
| 仅此一次、最后一次 | 虚假限时紧迫感 |
| 最后机会、错过不再有 | 虚假稀缺性 |
| 原价 XXX 元（无真实原价依据） | 虚构原价 |
| 直降 XX%（无真实折扣依据） | 虚假折扣幅度 |
| 免费领取（实为有条件） | 隐瞒限制条件 |

---

## 四、其他高风险词汇

| 违规表述 | 涉及条款 | 说明 |
|----------|----------|------|
| 国家级、国家认定（无依据） | 第九条 | 不得使用无依据的国家背书 |
| 荣获国际大奖（无具体来源） | 第二十八条 | 虚假荣誉宣传 |
| 专家推荐、医生建议（无资质） | 第三十八条 | 无资质代言 |
| 儿童必备、宝宝专用（诱导未成年人） | 第四十条 | 不当针对未成年人 |
