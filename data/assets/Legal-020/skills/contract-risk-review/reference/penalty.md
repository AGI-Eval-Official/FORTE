# 违约金

## 风险逻辑

违约金条款约定了一方违约时应支付的赔偿金额。不合理的违约金可能导致：
- 违约金过高，违约方承担不成比例的惩罚
- 违约金过低，守约方无法获得充分救济
- 违约金条款不对等，一方严苛一方宽松

## 法律依据

- 《中华人民共和国民法典》第五百八十五条：当事人可以约定一方违约时应当根据违约情况向对方支付一定数额的违约金，也可以约定因违约产生的损失赔偿额的计算方法。约定的违约金低于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以增加；约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。
- 司法实践中，违约金超过造成损失的30%一般认定为"过分高于"（参考已废止的《最高人民法院关于适用〈中华人民共和国合同法〉若干问题的解释（二）》第二十九条，该30%标准虽随解释废止而不再具有规范效力，但在现行《民法典》框架下，法院仍常以此为参考基准，最终以实际损失为基础，兼顾合同履行情况、当事人过错程度等综合判断）。

## 判定标准

### 甲方视角（委托方/采购方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 甲方违约金比例过高（如日息超过0.1%或总额超过合同金额30%） |
| 高 | 甲方违约金条款严苛而乙方违约金条款宽松或缺失（严重不对等） |
| 高 | 乙方无违约金条款或违约金极低（乙方违约成本过低） |
| 中 | 甲方违约金总额超过合同金额的20%但未超过30% |
| 中 | 双方违约金条款不完全对等但差异不大 |
| 低 | 双方违约金条款对等，比例合理 |

### 乙方视角（开发方/服务方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 乙方违约金比例过高（如日息超过0.1%或总额超过合同金额30%） |
| 高 | 乙方违约金条款严苛而甲方违约金条款宽松或缺失（严重不对等） |
| 高 | 甲方无违约金条款或违约金极低（甲方违约成本过低，如逾期付款无罚则） |
| 中 | 乙方违约金总额超过合同金额的20%但未超过30% |
| 中 | 双方违约金条款不完全对等但差异不大 |
| 低 | 双方违约金条款对等，比例合理 |

### 中立视角

| 等级 | 判定条件 |
|------|---------|
| 高 | 违约金条款严重不对等（一方有另一方无，或比例差异超过3倍） |
| 高 | 违约金总额超过合同金额的30%（参考司法实践标准） |
| 中 | 违约金条款存在但不够对等 |
| 中 | 违约金比例在合理范围内但计算方式不清晰 |
| 低 | 双方违约金条款对等，比例合理（如总额不超过合同金额的20%-30%） |

## 合同类型侧重

| 合同类型 | 权重 | 特有风险点 |
|---------|------|-----------|
| 软件开发 | 一般 | 逾期交付违约金与需求变更的关系、乙方交付延迟但因甲方需求变更导致 |
| SaaS订阅 | 低频 | SaaS 多以 SLA 违约补偿替代违约金 |
| 采购 | 重点 | 逾期交付违约金、质量不合格违约金、逾期付款违约金 |
| 技术许可 | 一般 | 超范围使用违约金、许可费逾期违约金 |
| 通用 | 标准 | — |

## 关键词与同义模式

### 核心关键词
- 违约金、滞纳金、逾期

### 扩展关键词
- 赔偿金、惩罚性、约定损失
- 日息、每日、按日计算
- 比例、百分比、万分之
- 违约责任、违约处罚
- 解约金、提前终止违约金

### 同义模式
- `每日.*%.*逾期/违约/滞纳`
- `违约金.*不超过.*合同.*金额.*%`
- `逾期.*天.*支付.*违约金`
- `违约.*赔偿.*倍`
- `一方.*违约.*另一方.*无违约金`
