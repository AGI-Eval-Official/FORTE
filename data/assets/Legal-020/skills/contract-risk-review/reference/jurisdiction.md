# 管辖法院

## 风险逻辑

管辖法院条款决定了争议解决的地点和适用程序，直接影响：
- 维权成本（异地诉讼的交通、住宿、律师费用）
- 司法环境差异（地方法院对本地企业的倾向性）
- 维权便利性和时效性

## 法律依据

- 《中华人民共和国民事诉讼法》第三十五条：合同或者其他财产权益纠纷的当事人可以书面协议选择被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地等与争议有实际联系的地点的人民法院管辖，但不得违反本法对级别管辖和专属管辖的规定。
- 协议管辖有效，但必须与争议有实际联系，且不得违反级别管辖和专属管辖规定。

## 判定标准

### 甲方视角（委托方/采购方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 管辖法院固定为乙方所在地（异地维权成本高） |
| 高 | 约定仲裁且仲裁机构在乙方所在地 |
| 中 | 管辖法院虽非甲方所在地但距离较近（如同城不同区） |
| 中 | 约定合同签订地管辖但签订地由乙方控制 |
| 低 | 管辖法院在甲方所在地或双方均可接受的地点 |
| 低 | 管辖法院约定为原告所在地（双方平等） |

### 乙方视角（开发方/服务方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 管辖法院固定为甲方所在地（异地维权成本高） |
| 高 | 约定仲裁且仲裁机构在甲方所在地 |
| 中 | 管辖法院虽非乙方所在地但距离较近 |
| 中 | 约定合同签订地管辖但签订地由甲方控制 |
| 低 | 管辖法院在乙方所在地或双方均可接受的地点 |
| 低 | 管辖法院约定为原告所在地（双方平等） |

### 中立视角

| 等级 | 判定条件 |
|------|---------|
| 高 | 管辖法院固定为一方所在地，明显增加对方维权成本 |
| 中 | 约定仲裁但仲裁机构不在双方所在地，增加双方维权成本 |
| 中 | 管辖法院约定在第三方地点但交通便利 |
| 中 | 管辖法院约定为合同签订地但签订地可控性存疑 |
| 低 | 管辖法院约定为原告所在地或合同履行地（双方平等） |
| 低 | 管辖法院约定为双方均可接受的中间地点 |

## 合同类型侧重

| 合同类型 | 权重 | 特有风险点 |
|---------|------|-----------|
| 软件开发 | 一般 | 合同履行地的认定（开发方所在地还是委托方所在地） |
| SaaS订阅 | 一般 | 服务器所在地是否构成"合同履行地" |
| 采购 | 一般 | 交货地与合同履行地的关系 |
| 技术许可 | 一般 | 许可技术使用地是否构成"合同履行地" |
| 通用 | 标准 | — |

## 关键词与同义模式

### 核心关键词
- 争议解决、管辖、诉讼、法院

### 扩展关键词
- 仲裁、仲裁委员会、仲裁机构
- 被告住所地、原告住所地、合同履行地、合同签订地
- 起诉、应诉、管辖权
- 异议、管辖权异议

### 同义模式
- `管辖.*法院.*甲方/乙方.*所在地`
- `向.*方所在地.*法院.*起诉`
- `争议.*提交.*仲裁`
- `由.*人民法院管辖`
- `合同签订地.*管辖`
