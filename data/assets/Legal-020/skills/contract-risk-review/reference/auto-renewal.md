# 自动续约

## 风险逻辑

自动续约条款使合同到期后自动延长，可能导致：
- 一方在不知情或遗忘的情况下被自动续约，继续承担合同义务
- 续约条件可能不利于当前市场情况（如价格未调整）
- 退出机制不明确，解除合同困难

## 法律依据

- 《中华人民共和国民法典》第五百六十三条：当事人一方迟延履行债务或者有其他违约行为致使不能实现合同目的，当事人可以解除合同。
- 自动续约条款的效力在司法实践中通常予以认可，但要求续约条件明确、退出机制合理。
- 部分行业监管规定对自动续约有特别要求（如电信、金融等需明确告知续约条件和退出方式）。

## 判定标准

### 甲方视角（委托方/采购方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 合同到期自动续约且无退出/解约机制 |
| 高 | 自动续约的退出窗口期过短（少于30天）或不易操作 |
| 高 | 自动续约后价格可由乙方单方调整 |
| 中 | 自动续约但需提前书面通知退出（通知期30-60天） |
| 中 | 自动续约的续约期限不明确或过长 |
| 低 | 续约需双方协商一致，不自动续约 |

### 乙方视角（开发方/服务方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 甲方有权在续约前单方终止且不承担任何费用 |
| 高 | 续约时甲方有权单方降低价格或缩减范围 |
| 中 | 自动续约但甲方可在较长时间内（超过60天）无理由退出 |
| 中 | 续约价格未约定，需重新协商但无保底价格 |
| 低 | 自动续约条款清晰，退出机制明确，续约条件对双方公平 |

### 中立视角

| 等级 | 判定条件 |
|------|---------|
| 高 | 自动续约但无退出机制或退出机制极不明确 |
| 高 | 续约条件完全由单方决定 |
| 中 | 自动续约附有退出机制但不够清晰 |
| 中 | 续约期限或价格约定不明 |
| 低 | 自动续约条款完整：续约条件明确、退出机制清晰、通知义务对等 |

## 合同类型侧重

| 合同类型 | 权重 | 特有风险点 |
|---------|------|-----------|
| 软件开发 | 低频 | 软件开发合同通常为一次性项目，较少涉及自动续约 |
| SaaS订阅 | 重点 | 自动续费是 SaaS 合同的核心风险，需特别关注退出窗口、续费价格、降级/升级选项 |
| 采购 | 低频 | 框架采购协议可能涉及续约，但通常需重新议价 |
| 技术许可 | 低频 | 部分许可涉及续期，关注续期价格和续期条件 |
| 通用 | 标准 | — |

## 关键词与同义模式

### 核心关键词
- 续约、续订、自动、期满

### 扩展关键词
- 续费、展期、延长、顺延
- 到期、届满、终止
- 退出、解约、取消、不续约
- 提前通知、书面通知
- 默认续期、自动展期

### 同义模式
- `到期.*自动.*续约/续费/展期/延长`
- `期满.*未.*通知.*视为.*续约`
- `提前.*天.*书面通知.*不续约/终止`
- `默认.*续期/续约`
- `续约.*价格.*调整/不变`
