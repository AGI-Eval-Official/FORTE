# 责任上限

## 风险逻辑

责任上限条款限制了一方在违约时的最大赔偿金额。不合理的责任上限可能导致：
- 受损方无法获得充分赔偿，实际损失远超赔偿上限
- 违约方的违约成本过低，缺乏履约约束
- 排除间接损失/利润损失的赔偿，使受损方无法获得完整赔偿

## 法律依据

- 《中华人民共和国民法典》第五百八十四条：当事人一方不履行合同义务或者履行合同义务不符合约定，造成对方损失的，损失赔偿额应当相当于因违约所造成的损失，包括合同履行后可以获得的利益。
- 《中华人民共和国民法典》第五百八十五条：当事人可以约定一方违约时应当根据违约情况向对方支付一定数额的违约金，也可以约定因违约产生的损失赔偿额的计算方法。约定的违约金过分高于造成的损失的，人民法院或者仲裁机构可以根据当事人的请求予以适当减少。
- 责任上限条款在司法实践中一般有效，但不得排除故意或重大过失责任（《民法典》第五百零六条）。

## 判定标准

### 甲方视角（委托方/采购方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 乙方赔偿责任上限明显偏低（低于合同总金额的50%） |
| 高 | 乙方仅退还已付费用，无额外赔偿 |
| 高 | 乙方排除间接损失、利润损失、数据损失的赔偿责任 |
| 高 | 乙方对数据丢失/泄露等重大风险不承担赔偿责任 |
| 中 | 乙方赔偿责任上限为合同总金额的50%-100% |
| 中 | 乙方排除间接损失赔偿但保留直接损失赔偿 |
| 低 | 乙方赔偿责任上限不低于合同总金额，赔偿范围合理 |

### 乙方视角（开发方/服务方）

| 等级 | 判定条件 |
|------|---------|
| 高 | 乙方赔偿责任无上限或被要求承担无限责任 |
| 高 | 甲方要求排除乙方的责任上限保护 |
| 中 | 责任上限未排除间接损失但乙方赔偿责任上限偏低（如低于合同总金额） |
| 中 | 甲方排除间接损失赔偿但乙方未获同等保护 |
| 低 | 双方责任上限对等，赔偿范围明确 |
| 低 | 责任上限为合同总金额，排除间接损失（行业惯例） |

### 中立视角

| 等级 | 判定条件 |
|------|---------|
| 高 | 责任上限条款严重不对等（一方受限一方无限制） |
| 高 | 完全排除间接损失且未提供替代保障 |
| 中 | 责任上限偏低但非极端不合理 |
| 中 | 排除间接损失赔偿但有其他补偿机制 |
| 低 | 双方责任上限对等，赔偿范围合理且明确 |

## 合同类型侧重

| 合同类型 | 权重 | 特有风险点 |
|---------|------|-----------|
| 软件开发 | 一般 | 软件缺陷导致的业务损失是否在赔偿范围内 |
| SaaS订阅 | 重点 | 服务中断/数据丢失的赔偿限制、SLA违约的赔偿上限 |
| 采购 | 一般 | 产品质量问题的赔偿范围、人身伤害是否排除在责任上限之外 |
| 技术许可 | 一般 | 许可技术缺陷导致的损失赔偿限制 |
| 通用 | 标准 | — |

## 关键词与同义模式

### 核心关键词
- 责任限制、赔偿责任、累计赔偿、上限、间接损失

### 扩展关键词
- 免责、不可抗力、直接损失、利润损失、数据损失
- 最高赔偿、赔偿总额、累计金额
- 退还、返还、补偿
- 故意、重大过失、欺诈

### 同义模式
- `赔偿.*不超过.*合同.*金额.*%`
- `仅.*退还.*已付费用`
- `不承担.*间接损失/利润损失/数据损失`
- `累计赔偿.*不超过`
- `责任.*限于.*直接损失`
