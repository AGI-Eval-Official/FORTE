# 案件信息字段定义

以下为默认字段定义。用户另有指定的字段或填写要求时，优先遵循用户指定，未指定的部分按以下默认定义执行。

## 一、基本信息

| 字段 | 说明 | 填写要求 |
|------|------|----------|
| 案号 | 唯一标识 | 完整案号，如 (2023)京0105民初12345号 |
| 案例名称 | 当事人信息概要 | 格式：XX诉XX+案由+案，如 张三诉李四买卖合同纠纷案 |
| 裁判年份 | 判决/裁定时间 | 四位年份 |
| 审理法院 | 法院名称及层级 | 需标注层级（基层/中级/高级/最高），如 北京市朝阳区人民法院（基层法院） |
| 当事人身份 | 原告/被告/第三人 | 需标注各方角色和简要身份，如 原告张三（供货方）/ 被告李四（收货方） |
| 案由 | 案件性质分类 | 使用法院认定的标准案由名称 |

## 二、案件核心内容

| 字段 | 说明 | 填写要求 |
|------|------|----------|
| 核心争议点 | 双方主要分歧所在 | 用一句简明的话概括，如 被告未付款是否构成根本违约 |
| 法院认定关键事实 | 法院查明的重要事实 | 提炼法院在"本院查明"部分认定的事实，保留关键细节 |
| 判决依据 | 法律条款、司法解释、相关政策 | 需标注具体法条，如《民法典》第577条、第579条 |
| 判决结果 | 支持/驳回/部分支持等 | 需写明具体判项内容，如 支持原告诉讼请求，判令被告支付货款50万元及违约金5万元 |

## 三、填写示例

```
案号：(2022)沪01民终9876号
案例名称：A公司诉B公司技术服务合同纠纷案
裁判年份：2022
审理法院：上海市第一中级人民法院（中级法院）
当事人身份：上诉人A公司（服务提供方）/ 被上诉人B公司（服务接受方）
案由：技术服务合同纠纷
核心争议点：B公司以服务成果未达标为由拒绝付款是否成立
法院认定关键事实：A公司已按合同约定交付技术开发成果，B公司使用该成果上线运营超过6个月且未在约定期限内提出书面异议
判决依据：《民法典》第577条、第579条；《最高人民法院关于审理技术合同纠纷案件适用法律若干问题的解释》第15条；《关于促进技术成果转化的若干规定》
判决结果：驳回上诉，维持原判；B公司支付技术服务费80万元及逾期付款违约金
```
